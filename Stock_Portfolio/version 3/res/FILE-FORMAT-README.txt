Strategy format:
1)Basic buy:
Unique identification number,Strategy class name, Ticker Symbol, Price, Commission, Date

2)Weighted Buy:
Unique identification number, Strategy class name, price,commission, Date, number of stocks(n), Ticker Symbol(1),.....Ticker Symbol(n), Weight(1),.....,Weight(n)

3)Dollar Cost Averaging:
Unique identification number, Strategy class name, price commission, Start date, End Date, Frequency, number of stocks(n), Ticker Symbol(1),.....Ticker Symbol(n), Weight(1),.....,Weight(n)
(Each strategy should occupy one line)

Portfolio format:
Unique Portfolio name, Ticker Symbol, Price, Commission, Date bought, Number of shares(Repeat for other stocks in the portfolio. All the portfolio data should be in one line)
